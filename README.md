* Internet Connection: A stable connection for the initial download and model syncing.

## 🚀 How to Generate Media

The interface centers on simple text prompts. You describe what you want to see, and the engine creates the result.

### Generating Images
1. Click the Image tab.
2. Type a description of your scene in the text box.
3. Adjust the style settings like aspect ratio or resolution.
4. Click the Generate button.
5. The image appears in your gallery folder once finished.

### Generating Videos
1. Click the Video tab.
2. Enter a detailed description of the motion you want to see.
3. Choose a video model from the dropdown menu.
4. Set the frame rate and video duration.
5. Press the Generate button.
6. The software processes the frames and saves the final file.

## 🔍 Frequently Asked Questions

What if the generation fails?
Check your storage space. High-resolution generations require significant temporary space while the software processes the files. Ensure your NVIDIA drivers remain current.

Can I turn off the fan monitor?
Yes, find this setting in the performance tab. Only disable this if you have advanced cooling on your graphics card.

How do I clear the cache?
Navigate to Settings, then click Preferences. Select Clear Cache to remove temporary files and free up disk space.

Does the software support custom models?
Yes. You can import models from other sources into the models folder. Follow the file naming convention found in the documentation folder within the installation directory.

Can I pause a generation?
Yes, use the Pause button on the progress bar. You can resume the task later without losing your current progress.

What happens if I lose internet access?
The software continues to function offline. You only need the internet to download new base models. All generation occurs on your hardware.
